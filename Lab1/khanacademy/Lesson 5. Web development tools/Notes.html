1. Using the browser development tools: you can inspect HTML and CSS of a site in the browser using a combination of keys.

2. Developing webpages outside of Khan Academy: you can use online editors, command-line editors, and desktop editors. The most popular editor is VS Code.

3. Hosting your website on a server: search the internet for "hosting providers" and "domain name registrars".
